# Agent Onboarding

This repository is now a basic React application demonstrating client-side data persistence with IndexedDB.

## How to self-orient
- Read `docs/introduction.md` to get the high-level overview of the platform.
- **Consult `project-manager/general-objectives.md` for the overarching goals of the project.**
- **Refer to `project-manager/styling-guide.md` for UI/UX principles and design consistency.**
- When adding or updating UI features, cross-check that every interaction traces back to the documented behavior.
